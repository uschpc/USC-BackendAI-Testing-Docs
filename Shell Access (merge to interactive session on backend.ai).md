# Shell access guide

Shell access for a compute session is available via a web-based terminal app.

## Accessing the terminal app

Once a session is started and running, follow these steps to access the terminal app:

1. In the **Sessions** list, locate your running session.
2. Click on the **Session Name** for the session.
3. A **Session Info** page will open.
4. Click on the terminal app icon **`>_`** in the upper right.
5. A new browser tab will open with the terminal interface.

## Using the terminal app

Within the terminal app:

- You will see a shell prompt (e.g., `work@main1[gDz6uH2p-session]:~$`).
- You can:
  - Run Linux shell commands (`ls`, `cd`, `python`, etc.)
  - Create and manage files
  - Install software

Closing the terminal browser tab does not stop the command that is running or the compute session.
